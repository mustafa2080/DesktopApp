# 🚀 دليل سريع - ميزة الغرف المشتركة

## ✅ تم التطبيق بنجاح!

تم إضافة حقول جديدة لجدول المعتمرين في قاعدة البيانات:
- ✅ **RoomType** - نوع الغرفة
- ✅ **SharedRoomNumber** - رقم الغرفة المشتركة
- ✅ **Index** على SharedRoomNumber لتسريع البحث

---

## 🎯 كيفية الاستخدام

### 1️⃣ في نموذج إضافة الحزمة:

```
┌────────────────────────────────────────────────────────┐
│ 📋 أسماء المعتمرين                                    │
├──────┬──────────────┬────────────┬──────────────┐
│ رقم  │ اسم المعتمر │ نوع الغرفة │ رقم الغرفة   │
├──────┼──────────────┼────────────┼──────────────┤
│  1   │ أحمد محمد    │ ثنائي ▼   │ R001         │ ← في نفس الغرفة
│  2   │ محمود علي    │ ثنائي ▼   │ R001         │ ← في نفس الغرفة
│  3   │ خالد حسن     │ مفردة ▼   │ R002         │ ← غرفة منفردة
└──────┴──────────────┴────────────┴──────────────┘
```

### 2️⃣ خيارات نوع الغرفة:
- 🛏️ مفردة
- 🛏️🛏️ ثنائي (افتراضي)
- 🛏️🛏️🛏️ ثلاثي
- 🛏️🛏️🛏️🛏️ رباعي
- 🛏️🛏️🛏️🛏️🛏️ خماسي

### 3️⃣ رقم الغرفة:
- اكتب **نفس الرقم** للمعتمرين المشتركين في غرفة واحدة
- مثال: R001, R002, R003
- الحقل **اختياري** - يمكن تركه فارغاً

---

## 📊 أمثلة عملية

### مثال 1: عائلة في غرف منفصلة
```
1. أحمد (رب الأسرة)     → مفردة    → R101
2. فاطمة (زوجته)         → مفردة    → R102
3. محمد (ابن)            → ثنائي    → R103
4. علي (ابن)             → ثنائي    → R103
```

### مثال 2: مجموعة أصدقاء
```
1. أحمد    → ثنائي → R201
2. محمود   → ثنائي → R201
3. خالد    → ثنائي → R202
4. عمر     → ثنائي → R202
5. سعيد    → ثنائي → R203
6. علي     → ثنائي → R203
```

### مثال 3: مجموعة كبيرة
```
1. معتمر 1  → ثلاثي → R301
2. معتمر 2  → ثلاثي → R301
3. معتمر 3  → ثلاثي → R301
4. معتمر 4  → رباعي → R302
5. معتمر 5  → رباعي → R302
6. معتمر 6  → رباعي → R302
7. معتمر 7  → رباعي → R302
```

## 🔍 فوائد الميزة

✅ **تنظيم أفضل** - معرفة من يشترك مع من في الغرفة
✅ **تقارير دقيقة** - إحصائيات عن الغرف المستخدمة
✅ **سهولة البحث** - إيجاد جميع المعتمرين في غرفة محددة
✅ **مرونة** - يمكن تعديل التوزيع في أي وقت

---

## 📝 ملاحظات مهمة

⚠️ **نوع الغرفة يجب أن يتطابق مع عدد المعتمرين في نفس الغرفة**

مثال صحيح ✅:
```
أحمد → ثنائي → R001
محمود → ثنائي → R001
(2 معتمرين في غرفة ثنائية)
```

مثال خاطئ ❌:
```
أحمد → ثنائي → R001
محمود → ثنائي → R001
خالد → ثنائي → R001
(3 معتمرين في غرفة ثنائية!)
```
